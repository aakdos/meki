# MEKI

> Discord Monster Collection RPG

## 프로젝트 소개

MEKI는 디스코드에서 플레이하는 수집형 RPG 게임입니다.

플레이어는 몬스터를 수집하고, 팀을 편성하며,
지역 시너지와 태생 시너지를 활용하여
다양한 던전과 보스를 공략합니다.
