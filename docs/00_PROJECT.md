# MEKI Project

> Discord 기반 몬스터 수집 · 육성 RPG

---

# 프로젝트 소개

**MEKI**는 Discord에서 플레이하는 수집형 RPG 게임입니다.

플레이어는 몬스터를 수집하고, 육성하고, 팀을 구성하여 다양한 콘텐츠를 공략합니다.

현재는 **메이플스토리의 몬스터를 기반으로 프로토타입을 개발**하며, 게임 시스템이 안정화되면 모든 IP를 오리지널 콘텐츠로 교체하는 것을 목표로 합니다.

---

# 프로젝트 철학

MEKI는 단순히 희귀 몬스터를 뽑는 게임이 아닙니다.

**좋은 개체를 성장시키고, 다양한 시너지를 연구하며, 자신만의 덱을 완성하는 게임**을 목표로 합니다.

낮은 등급의 몬스터도 충분히 가치가 있어야 하며, 플레이어의 전략이 결과를 바꿀 수 있어야 합니다.
